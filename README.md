

## Installation

```bash
$ nest u

// 需額外安裝以下
$ npm i --save @nestjs/graphql apollo-server-express graphql-tools graphql type-graphql
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## open GraphQL

```bash
http://localhost:3000/graphql
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
